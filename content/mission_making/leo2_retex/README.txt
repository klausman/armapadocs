Desert/Woodland for the Arma3 Leopard 2 (aka 2SG aka Revolution).

Place object as normal, add code below to init.

Note that the camo nets are *not* retextured.

Desert:

```
this setVariable ["BIS_enableRandomization", false];
this setObjectTexture[0, "leo_dsrt_01.paa"];
this setObjectTexture[1, "leo_dsrt_02.paa"];
this setObjectTexture[2, "leo_dsrt_rcws.paa"];
```

Woodland:

```
this setVariable ["BIS_enableRandomization", false];
this setObjectTexture[0, "leo_dsrt_01.paa"];
this setObjectTexture[1, "leo_dsrt_02.paa"];
this setObjectTexture[2, "leo_dsrt_rcws.paa"];
```
